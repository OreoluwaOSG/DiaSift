Project: Diasift

Diasift is a web based Type 2 Diabetes guideline assistant that uses Retrieval Augmented Generation to answer general educational questions from trusted public sources such as NHS and NICE.

The system will not diagnose users, prescribe medicine, recommend personal treatment changes, or replace healthcare professionals.

Main features:
1. Search Type 2 Diabetes guidance documents.
2. Answer questions using retrieved evidence.
3. Show citations for each answer.
4. Label the evidence as Strong, Partial, or No clear evidence.
5. Refuse unsafe or unsupported personal medical questions.
